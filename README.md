# responsible-dashboard
Um dashboard de jogo responsável, guiado por IA avançada, capaz de transformar dados em transparência, controle e experiências melhores para o cliente.  A ideia é simples: usar tecnologia de ponta (React, Python, LangChain, AWS/Bedrock, Databricks) para construir algo que realmente faça diferença no dia a dia dos jogadores.
